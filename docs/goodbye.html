Saying good bye to Juli

As we sunset WebCatalog for Windows & Linux, most of Juli's codebase'll be merged with the main app in the next major release of WebCatalog. So we'll no longer maintain Juli as a seperate app.

You can continue to download and use the app normally. But it'll no longer receive any updates from us.

Thank you for using Juli.
